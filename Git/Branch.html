Branching in Git

Branching allows us to isolate changes

Change Directory to Git Repository

cd git/project

Checkout a New Branch

git checkout -b <branch_name>

If you can't create a new branch you may need to commit and push your changes first.

Alternatively, see Stashing.

git stash save --keep-index

Then try again

git checkout -b <branch_name>
