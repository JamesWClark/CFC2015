Git Stash

Typically used when switching between branches.

https://git-scm.com/book/en/v1/Git-Tools-Stashing

Two Use Cases for Stash

  1. When we have work in progress and need to switch branches to work on something else.
  2. When we hit a dead end and wish to scrap our current progress.

Work in Progress (Save)

git stash save

Dead End (Discard)

git stash drop
