Processing 2D

Sketch and animate 2D graphics in the Processing language. An introduction or review of programming fundamentals: variables, functions, and object oriented design.

The ultimate goal here is to build a game like Space Invaders. We won't create an exact copy but we will approach the concept. Before we can build a game like this, we must first develop a clear set of objectives. Here is a list of tasks we need to accomplish in code in order to make this game run:

  • Create game graphics
  • Interact with inputs (keyboard & mouse)
