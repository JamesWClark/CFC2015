Pixel Processing

Explore programmatic manipulation of images pixel by pixel. Apply photo effects: blending, transformation, tint, transparency, opacity, hue, saturation, brightness, erosion, dilation, posterization, greyscale, color inversion.

This is Pixel Processing. Before we can start manipulating pixels, we need to first discuss what pixels are and how they're stored in data. Also, for those attending for the first time this week, we need to get familiar with Processing.
